# Data Integration Planner

A small planner UI and utilities for designing data integration pipelines and schemas.

## Tech Stack

- Vite (bundler)
- React + TypeScript (UI)
- Tailwind CSS + PostCSS (styles)
- Supabase (serverless functions)
- Radix UI, MUI, and various React libs (UI components)

## Features

- Schema designer (src/app/pages/schema-designer.tsx)
- Realtime examples (src/app/pages/realtime.tsx)
- Caching and performance tips (src/app/pages/caching.tsx)
- Authentication examples (src/app/pages/authentication.tsx)
- API endpoint examples (src/app/pages/api-endpoints.tsx)
- Component library and UI primitives under `src/app/components/ui`

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## NPM Scripts

- `dev`: starts Vite development server
- `build`: builds the production bundle
